# Token Minimizer | Save Money for Education

This tool helps users reduce the cost of AI prompts (OpenAI, Claude, etc.) by converting text into high-density logic (Chinese + Simplified Grammar).

**Mission:** Stop wasting money on redundant tokens. Use the savings to invest in real human education.

## How to Deploy to GitHub Pages (Free Hosting)

1.  **Initialize Git** (if not done):
    ```bash
    git init
    git add .
    git commit -m "Initial commit of Token Minimizer"
    ```

2.  **Create a Repository on GitHub:**
    *   Go to [github.com/new](https://github.com/new).
    *   Name it `token-minimizer` (or similar).
    *   Do **not** add README/gitignore yet (you have them).

3.  **Push your code:**
    ```bash
    git remote add origin https://github.com/YOUR_USERNAME/token-minimizer.git
    git branch -M main
    git push -u origin main
    ```

4.  **Enable GitHub Pages:**
    *   Go to your repository **Settings** -> **Pages**.
    *   Under **Source**, select `Deploy from a branch`.
    *   Select `main` branch and `/ (root)` folder.
    *   Click **Save**.

Wait 1-2 minutes, and your site will be live at `https://YOUR_USERNAME.github.io/token-minimizer`.

## Features

*   **Nuclear Mode:** Strips all political correctness and filler words.
*   **Chinese Density:** Translates logic to Chinese (high token density) while instructing AI to reply in your language.
*   **Safety Net:** Auto-switches to English optimization if translation fails.
*   **.toon Format:** Save optimized prompts for later use.


## Usage
1. Open `index.html` in your web browser.
2. Select your input language (e.g., Spanish).
3. Type your prompt.
4. Click **Translate & Optimize**.
5. Copy the result or download as `.toon`.
